# Date.java

´Adaptado de K. N. King: C Programming: A Modern Approach, 2nd Edition.´

Elabore um programa que LÊ DO TECLADO uma data no formato dd/mm/aaaa e ESCREVE NA TELA a data no formato usual norte-americano, por extenso. 

Por exemplo, a data 13/09/2023 deve ser impressa como September 13th, 2023.

Lembre-se das regras dos ordinais em inglês.
