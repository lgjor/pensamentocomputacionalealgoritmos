# Exercício em sala: IMC.java

Elabore um programa que LÊ DO teclado dois números reais representando o peso e a altura de uma pessoa e ESCREVE NA TELA o índice de massa corpórea (IMC) desta pessoa, bem com a classificação do seu estado nutricional. Veja a página do Ministério da Saúde para mais informações.

## Restrições

- Não utilize a classe Math.

## Exemplos de execução: 1 2 3

```shell
Entre com o peso (kg): 83
Entre com a altura (m): 1.75
IMC: 27.10 kg/m2

Classificação: Sobrepeso
```
